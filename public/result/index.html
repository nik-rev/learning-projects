Result Cheatsheet

A cheatsheet on the Result type in Rust.

We use abstract names for types which can represent literally anything such as T and K.

Feel free to mentally substitute types which make more sense to you, like for example &str and i32

How to read this cheatsheet:

  • Some T

    Some type T wrapped in a Some

  • T ➝ K

    A closure which takes some type T as an argument and returns another type K

  • D, E ➝ Err N

    The first argument is of some type D, the second argument is a closure that takes a T as an argument and always returns some other type N wrapped in an Err

  • 💥

    Indicates an unsafe function or undefined behaviour
